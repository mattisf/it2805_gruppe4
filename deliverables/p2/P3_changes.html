Design Changes that are made

The page layout and appearance are followed as planned and used as a template for the pages. In addition, a footer is designed and implemented on every pages. The initially 10 planned pages have used the mockups as a suggestion of model of how it will look like and operate. Mostly all of them have the same fuctionality, but some paged has a new layout. In addition, new pages and functionalities has been developed and implemented. Concrete what changes, both functionals, visuals and the new pages themself, are presented and described below.

homePage.html

Homepage has a totally new layout. The navbar is the same as the rest of the sites, to improve the usability and smooth trantition when navigating to another page. The landingpage is implemented with inspiration pictures and quotes. This was decided in collaboration with the client.

ourHistory.html

The only change is the text color of the title "our history". The original color was light pink (#FCF6FC), like the background. The new color is black to improve the usability.

production.html

The only change is the text color of the title "product information". The original color was light pink (#FCF6FC), like the background. The new color is black to improve the usability.

contactUs.html

The first change is the text color of the title "contact us". The original color was light pink (#FCF6FC), like the background. The new color is black to improve the usability. In addition, the Twitter text below is excluded, when this is not an important platform for the client.

contactScheme.html

This page is added to the content. The page has a contact scheme where the user can write a message and its information to get in touch with the client. The page is the result from a request from the client, where they wanted to facilitate for and ease communication from customers.

shoppingCart.html

Shopping cart has a new layout, but the same functionalities. Each item is now represented on a card, including the name and information about the product, the order amount of the item and buttons to change it. The product image is styled to be: height: 25ch and width: 20ch. The page has a responsive price, both for each product, ordinary price and total price, including freight cost. The functionalites where the user can add a promo code, and get information about the shipping methods are also excluded. The checkout button is implemented with an alert instead of transferring the user to an external site. The alert, "Thank you for your order", gives a feedback on the order.
